jQuery Mobile Course Project
Project Description

This project is a mobile web application built using jQuery Mobile. The purpose of the application is to demonstrate mobile interface components such as lists, forms, navigation panels, grid layouts, and Web Storage functionality. The application allows users to navigate between pages, enter personal information, and store favorite items using browser localStorage.

Page Descriptions
Home Page

The home page acts as the main navigation page for the application.

Buttons:

Filtered List

Action: Opens the filtered list page

Result: Displays searchable list of items

User Form

Action: Opens the user information form

Result: Allows users to enter and save personal information

Grid Layout

Action: Opens the grid page

Result: Displays a 4-column responsive layout

Favorites

Action: Opens the favorites page

Result: Allows the user to add and view favorite items stored in Web Storage

Filtered List Page

This page displays a searchable list using the jQuery Mobile listview component.

Buttons / Features:

Search Filter

Allows the user to filter list items in real time.

Favorite Item Icon

Displays an icon next to a list item.

Back Button

Returns the user to the home page.

User Form Page

Buttons:
